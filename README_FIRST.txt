## 🌍 Optional: ServiceNow Integration

To use real ServiceNow incidents:

1. Edit `backend/.env`
2. Add your credentials:
```
SERVICENOW_INSTANCE=your-instance.service-now.com
SERVICENOW_USERNAME=your_username
SERVICENOW_PASSWORD=your_password
```
3. Restart backend

**Without ServiceNow?** That's fine! The app works with sample data.

## 🆘 Quick Troubleshooting

| Problem | Solution |
|---------|----------|
| Port already in use | Change PORT in .env file |
| npm not found | Install Node.js |
| python not found | Install Python |
| Can't connect to backend | Make sure backend is running |
| CORS errors | Restart both servers |
